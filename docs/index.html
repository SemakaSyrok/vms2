Documentation

Documentation for Video Management System

Requirements

  • Node.js
  • Postgres

This app consists of 3 parts:

  • / - main(server) app
  • /client - admin app
  • /user - user app

- Structure

User app

It is simple Vue PWA app.

- Technologies

  • Vue
  • Vuex
  • Vue Router
  • Bootstrap

- Structure

  • /user/src/assets for your assets (styles, images...)
  • /user/src/components for your no-pages Vue components
  • /user/src/pages for your Vue pages
  • /user/src/router for Vue router
  • /user/src/services for your Validate Service
  • /user/src/store for Vuex
  • /user/src/App.vue entry point


Admin app

It is simple Vue PWA app.

- Technologies

  • Vue
  • Vuex
  • Vue Router
  • Bootstrap

- Structure

  • /user/src/assets for your assets (styles, images...)
  • /user/src/components for your no-pages Vue components
  • /user/src/pages for your Vue pages
  • /user/src/router for Vue router
  • /user/src/services for your Validate Service
  • /user/src/store for Vuex
  • /user/src/App.vue entry point


Server app

It is Node.js app

- Technologies

  • Node.js
  • Express
  • Sequelize
  • Postgres

Configure your .env (default ports for admin and client app is 3001, if you want to to change it, change it also in /user/config/dev.env.js and /client/config/dev.env.js)

- Structure

  • /uploads for uploads
  • /steps for uploads
  • /index.js entry point
  • /io.js for sockets
  • /migrations.js for your migrations
  • /controllers for your Controllers
  • /models for database and models
  • /usages.jsExpress middleware

Build your user and admin app (npm run build)
Main admin is user with id=1

Turn to true for migrate


Run your app



Go to http://localhost:3001/admin



Type login and password from migrations



Congratulations, You are in the app!
